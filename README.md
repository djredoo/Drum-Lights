# Drum-Lights
### Drum lighting I created for my marching band based on an Arduino Pro Mini.
---
Materials:
+ 1x    Arduino Pro Mini
+ 1x    TIP31C
+ 1x    330 Ohms Resistor
+ 1x    1 MOhms Resistor
+ 1x    Piezosensor
+ 1-2x  Battery Clip (9v)
+ 1x    LED-Strips (12V)
